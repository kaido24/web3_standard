Web3 standard installation profile

=============
About
=============
This is Drupal 7 standard installation profile with some extra features.
Extra modules:
Admin menu
Admin views
Wysiwyg
Pathauto
Views
Global redirect
XML Sitemap
some others.
Extra settings:
Wysiwig settings
Default format is full html with ckeditor

=============
Istallation
=============
Add these files to profile folder and copy drupal-org.make and local.make root directory of your planned drupal folder.
Run command: drush make local.make
